# pdf_merge

**Installation**
```
# clone the repo
$ git clone https://github.com/Moritz1404/pdf_merge.git

# change the working directory to pdf_merge
$ cd pdf_merge

# install the requirements
$ python3 -m pip install -r requirements.txt

# run the program
$ python3 main.py
```
